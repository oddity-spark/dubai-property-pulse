# Dubai Property Pulse

A portfolio analytics project using Dubai Land Department sales transaction data to identify residential sales momentum across Dubai communities.

## Business question

Which Dubai communities show the strongest residential sales momentum based on transaction volume, median price per sqm, quarterly growth, and off-plan vs ready sales mix?

## V1 scope

- Sales transactions only
- 2023–2025 or latest available 24–36 months
- Residential apartments first
- Top 20–30 communities
- Quarterly transaction volume
- Median price per sqm
- QoQ growth
- Off-plan vs ready share

## Tools

- Python for cleaning
- PostgreSQL for analysis
- Dashboard tool to be decided
